# 注音朗讀練習工具 設計文件

**日期**：2026-02-26

## 概述

在現有魔術方塊盲解訓練 App 中新增「隨機朗讀注音」功能。工具會從用戶的記憶詞字典（484 個注音配對）中隨機抽取，自動語音朗讀注音符號，讓用戶有時間反應，再揭曉記憶詞，無限循環。

## 功能需求

- 484 個配對（22×22 注音符號組合），每輪隨機打亂
- 播完一輪後自動重新隨機，無限循環
- 朗讀內容：注音配對（如「ㄍ ㄌ」）→ 停頓 → 記憶詞（如「老公」）
- 間隔時間可調：「問題間隔」和「答案停留」各自獨立設定（秒數）
- 讀取用戶自訂記憶詞字典（從 CubeContext 取得）
- 正常前景播放（不需要背景/螢幕關閉後繼續）

## 技術方案

**語音**：Web Speech API（`SpeechSynthesis`），使用中文語音，離線可用

**入口**：`FloatingButtons` 新增喇叭按鈕

**Modal**：全螢幕 Portal（參考現有 `EncodingPanel` 做法）

## UI 設計

### 風格
- 背景：`from-slate-900 via-slate-800 to-slate-900`（與 App 一致）
- glassmorphism header
- 注音大字：72px 白色
- 記憶詞：48px，`???` 用 `text-slate-400`，答案 fade-in 200ms
- 進度條：indigo-500

## 播放流程

1. 從打亂的 484 配對陣列取下一個
2. 語音唸出兩個注音（「ㄍ」「ㄌ」，中間略停）
3. 畫面顯示注音 + `???`，倒數「問題間隔」秒
4. 語音唸出記憶詞（「老公」）
5. 畫面顯示記憶詞，停留「答案停留」秒
6. 進入下一個配對
7. 484 個播完後重新打亂，繼續循環
